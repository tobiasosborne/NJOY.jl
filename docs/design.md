# Historical Design Proposal

The original pre-implementation design proposal has been archived verbatim at
[`archive/design-original.md`](archive/design-original.md).

That proposal is historical context, not current project guidance. Several of
its goals and constraints were superseded as oracle work established the
requirements for a faithful NJOY2016 replacement, including its in-memory
module flow, differentiability goals, scratch-tape prohibition, validation
tolerances, and matrix/Faddeeva implementation choices.

Use these sources for current guidance:

- [`../AGENTS.md`](../AGENTS.md) — governing laws and workflow;
- [`architecture.md`](architecture.md) — current deck/tape architecture;
- [`../HANDOFF.md`](../HANDOFF.md) — live implementation state and open work;
- [`../reports/ACCEPTANCE_CRITERIA.md`](../reports/ACCEPTANCE_CRITERIA.md) —
  numeric and structural acceptance rules.
